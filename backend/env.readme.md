## ENV FILE REQUIREMENTS
### POSTGRESQL
- **host**
    - **PG_HOST**
      - VALUE
          - in local
              - `postgresql` (docker container name)
              - `host.docker.internal`
              - `localhost`
- **port**
    - **PG_PORT**
- **username**
    - **PG_USER_NAME**
- **password**
    - **PG_PASSWORD**
- **DB Name**
    - **PG_DATABASE**
